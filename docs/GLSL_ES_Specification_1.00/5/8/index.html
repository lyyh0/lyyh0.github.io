GLSL ES Specification 1.00----5.8 赋值

返回目录

使用赋值运算符（=）将值赋值给变量名：

  lvalue-expression = rvalue-expression

该值表达式的计算结果为l值。赋值运算符将右值表达式的值存储到l值中，并返回一个具有左值表达式类型和精度的r值。左值表达式和右值表达式必须具有相同的类型。所有需要的类型转换都必须通过构造函数显式指定。内置类型的变量、整个结构、结构字段、应用字段选择器（.）选择组件或无重复字段的swizzle的l值、括号内的l值以及用数组下标运算符（[ ]）解引用的l值都是l值。其他一元、二元和三元表达式不是l值。这包括赋值、函数名、具有重复字段和常量的swizzle。

数组变量是l值，可以传递给声明为out或inout的参数。但是，它们不能用作任务的目标。同样，包含数组的结构可以传递给声明为out或inout的参数，但不能用作赋值的目标。

  • 算术作业加（+=），减（-=），乘（*=），除（/=）。表达

      lvalue-expression op= expression

    等效于

      lvalue = lvalue op expression

    其中，左值是lvalue-expression返回的对象lvalue-expression只计算一次。l值和表达式必须满足op和相等（=）的语义要求。

  • 分配余数为（%=）、左移为（<<=）、右移为（>>=）、包含或为（|=）以及独占或为（^=）保留以供将来使用。

在写入（或初始化）变量之前读取变量是合法的，但该值未定义。
