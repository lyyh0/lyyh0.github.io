GLSL ES Specification 1.00----5.4.2 向量和矩阵构造器

返回目录

构造函数可用于从一组标量、向量或矩阵中创建向量或矩阵。这包括缩短矢量的能力。

如果向量构造函数只有一个标量参数，则用于将所构造向量的所有组件初始化为该标量的值。如果矩阵构造函数只有一个标量参数，则它用于初始化矩阵对角线上的所有组件，其余组件初始化为0.0。

如果一个向量是由多个标量、向量、矩阵或它们的混合构成的，那么向量的分量将从参数的分量开始按顺序构造。参数将从左向右被消耗，每个参数的所有组件都将按顺序被消耗，然后再消耗下一个参数的任何组件。类似地，用于从多个标量或向量或这些标量或向量的混合构建矩阵。矩阵组件将按列主顺序构建和使用。在这些情况下，参数中必须提供足够的组件，以便为构造值中的每个组件提供一个初始化器。在上次使用的参数之外提供额外的参数是错误的。

如果一个矩阵是由一个矩阵构造的，那么结果中每个在参数中有对应分量（列i，行j）的分量（列i，行j）都将从那里初始化。所有其他组件都将初始化为单位矩阵。如果向矩阵构造函数提供矩阵参数，则具有任何其他参数都是错误的。

如果构造函数的参数的基本类型（bool、int或float）与正在构造的对象的基本类型不匹配，则使用标量构造规则（如上所述）来转换参数。

一些有用的向量构造函数如下：

  vec3(float) // 用浮点数初始化的每个组件
  vec4(ivec4) // 使用分量转换生成vec4

  vec2(float, float) // 用2个浮点数初始化vec2
  ivec3(int, int, int) // 用3个整数初始化ivec3
  bvec4(int, int, float, float) // 使用4个布尔转换

  vec2(vec3) // 删除vec3的第三个组件
  vec3(vec4) // 删除vec4的第四个组件

  vec3(vec2, float) // vec3.x = vec2.x, vec3.y = vec2.y, vec3.z = float
  vec3(float, vec2) // vec3.x = float, vec3.y = vec2.x, vec3.z = vec2.y
  vec4(vec3, float)
  vec4(float, vec3)
  vec4(vec2, vec2)

其中一些例子是：

  vec4 color = vec4(0.0, 1.0, 0.0, 1.0);
  vec4 rgba = vec4(1.0); // 将每个组件设置为1.0
  vec3 rgb = vec3(color); // 放下第四个组件

要在所有其他元素都设置为零的情况下初始化矩阵的对角线，请执行以下操作：

  mat2(float)
  mat3(float)
  mat4(float)

通过指定向量或分别通过mat2、mat3和mat4的所有4、9或16个浮点数来初始化矩阵。浮点数按列主顺序分配给元素。

  mat2(vec2, vec2);
  mat3(vec3, vec3, vec3);
  mat4(vec4, vec4, vec4, vec4);

  mat2(float, float,
    float, float);

  mat3(float, float, float,
    float, float, float,
    float, float, float);

  mat4(float, float, float, float,
    float, float, float, float,
    float, float, float, float,
    float, float, float, float);

只要存在足够的组件来初始化矩阵，就存在广泛的其他可能性。
