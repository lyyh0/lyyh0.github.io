GLSL ES Specification 1.00----5.6 矩阵分量

返回目录

可以使用数组下标语法访问矩阵的组件。对矩阵应用单个下标会将矩阵视为列向量数组，并选择一列，该列的类型是与矩阵大小相同的向量。最左侧的列是列0。然后，第二个下标将对列向量进行操作，如前面为向量定义的那样。因此，两个下标选择一列，然后选择一行。

  mat4 m;
  m[1] = vec4(2.0); // 将第二列全部设置为2.0
  m[0][0] = 1.0; // 将左上角元素设置为1.0
  m[2][3] = 2.0; // 将第三列的第四个元素设置为2.0

访问矩阵边界外的组件时的行为与访问向量和数组时的行为相同。如果索引表达式是常量表达式，但非常量表达式的行为未定义，则编译器必须生成错误。（例如，mat3的组件[3][3]必须生成编译时错误）。
