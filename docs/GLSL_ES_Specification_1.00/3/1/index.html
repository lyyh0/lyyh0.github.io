GLSL ES Specification 1.00----3.1 字集

返回目录

用于OpenGL ES着色语言的源字符集是ASCII的一个子集（见第12节：“标准参考”）。它包括以下字符：

  字母a-z、a-z和下划线（_）。

  数字0-9。

  符号句点（.）、加号（+）、破折号（-）、斜线（/）、星号（*）、百分比（%）、尖括号（<和>）、方括号（[和]）、括号（(和)）、大括号（{和}）、插入符号（^）、竖线（|）、与号（&）、波浪号（~）、等号（=）、感叹号（!）、冒号（:）、分号（;）、逗号（,）和问号（?）

  预处理器使用的数字符号（#）。

  空白字符：空格字符、水平制表符、垂直制表符、换行符、回车符和换行符。

行与编译器诊断消息和预处理器相关。它们通过回车符或换行符终止。如果两者一起使用，它将仅算作单线端接。在本文档的其余部分，任何这些组合都简称为新行。

行连续字符（\）不是语言的一部分。

一般来说，语言对这个字符集的使用是区分大小写的。

没有字符或字符串数据类型，因此不包括引号字符。

没有文件结尾字符。源字符串的结尾由长度而不是字符指示给编译器。
