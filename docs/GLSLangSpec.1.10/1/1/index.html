GLSLangSpec.1.10----1.1 自1.051版本以来的更改

返回目录

  • 对于新增#101至#105问题，我们针对这些问题所做的规范更改方案是调整数组参数的大小，并在构造函数中添加一些限制。详情见第4.2、5.4.2、6.1、6.1.1节。
  • 添加了与ATI_draw_buffers和ARB_color_camp_control的交互，特别是输出变量gl_FragData[n]。
  • 3.3 添加了#version和#extension来声明版本和扩展。
  • 7.5 为矩阵的逆和转置添加了内置状态。
  • 8 添加了内置函数refract、exp和log。
  • 增加了以下澄清和更正：
    • 2.1 从顶点处理器的工作列表中删除“颜色夹紧”。这已经过时了。
    • 2.1 将“透视投影”改为更清楚地指出属于不同列表的投影变换和透视分割。
    • 3.3 保留的启动“GL_”的预处理器宏
    • 3.6 添加了保留字packed、this、interface、sampler2DRectShadow。并且澄清列出这些关键字和保留字是唯一的。
    • 4.1.5 删除“整数向量可用于从纹理读取中获取多个整数。”这已经过时了。
    • 4.3.5 明确了结构可以是常量，以及const必须初始化。
    • 5.8 明确*=、+=等等的真正含义。并且?:不是l值。
    • 5.9 明确整数和浮点数在标量和向量之间允许进行的操作，列表中列出了所有运算符和表达式。
    • 6.1 更正一些原来的示例。这些示例并没有被正确填写，而这些示例被纠正了一段时间。
    • 6.1 添加澄清“如果在调用内置函数之前，在着色器中重新声明了该函数（即原型可见），则链接器将仅尝试在与其链接的着色器集中解析该调用。”
    • 7.2 删除过时的文本“将实现在计算深度的着色器中提供一个具有相同源级表达式的恒定结果，但着色器和固定功能之间不提供不变性。”
    • 7.4 更正内置常量名称列表：删除后缀并使值保持最新。
    • 8.2 说明指数函数的域。
    • 8.3 更改step()以比较x<edge而不是x<=edge。
    • 8.7 澄清关于当阴影查找是未定义时。
    • 8.9 进一步指定噪声的范围和频率约束。
    • 语法：MOD_ASSIGN更改为保留（以匹配规范文本）。
    • 语法：更改为要求函数参数中的数组大小。
  • 修复了几个拼写错误。
